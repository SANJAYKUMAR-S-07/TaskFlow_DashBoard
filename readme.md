# To-Do List Web Application

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript. This application helps users manage daily tasks efficiently by allowing them to add, organize, track, and complete tasks through an intuitive interface.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Edit existing tasks
* Organize tasks into different categories
* Responsive user interface
* Clean and modern design
* Local storage support for saving tasks
* Real-time task updates

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Usage

### Add a Task

* Enter a task in the input field.
* Click the Add button.

### Edit a Task

* Click the Edit button next to the task.
* Modify the task and save changes.

### Complete a Task

* Click the checkbox or complete button.
* The task will move to the completed section.

### Delete a Task

* Click the Delete button to remove the task permanently.
